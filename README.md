# python-3D-maze
um projeto de um sistema labirinto 3d feito em python usando openGL [em andamento]
## como rodar?
execute o arquivo requirements.txt após isso execute o arquivo **main.py**

---

## Como baixar?
### baixe os arquivos desse git ou de no seu computador o comando
 ```bash
   git clone https://github.com/HeitorASM/python-3D-maze.git
 ```
### A apos isso **entre na pasta do arquivo abra o console e execute esse comando:**
```bash
pip install -r requirements.txt
```
---
## Requisitos

- **Python 3.6+**
- **Bibliotecas básicas:**  
  - OpenGL
  - Pillow
  - pygame==2.5.2
  - Numpy
- **Verçoes do Modolos:**
  - pygame==2.5.2
  - PyOpenGL==3.1.7
  - PyOpenGL_accelerate==3.1.7
  - numpy==1.24.3
  - Pillow==9.5.0
